# Domain Rollout

## Target domain

propertyquarry.com

## Rollout sequence

1. Point the domain at the existing property-capable runtime
2. Serve a dedicated host-based public shell
3. Keep application links domain-correct
4. Add product-specific analytics and billing hooks

## Early technical rule

The public host should shape:

- brand name
- landing copy
- pricing copy
- onboarding posture
- generated links in the product flow

## Deployment note

During the first extraction phase, the backend may remain shared with EA as long as:

- public branding is separated
- the commercial boundary is separated
- property users do not see executive-assistant framing
