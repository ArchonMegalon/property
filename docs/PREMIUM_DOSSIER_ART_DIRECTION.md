# Premium Dossier Art Direction

The dossier should read like a compact premium advisory packet, not a system receipt.

## Page Rhythm

```text
1. Cover with real property/media image or premium neutral fallback
2. Executive decision
3. Core facts
4. Tour and media links
5. Daily-life radius
6. Risk and missing facts
7. Investment or household read when relevant
8. Agent questions
9. Provenance and privacy
```

## Rules

```text
cover image or poster visible on page one
no artifact status tables
no raw URLs as visible text
no operator jargon
no more than four dense cards per page
facts before narrative claims
risks include next actions
unsupported claims fail QA
```

## Visual QA Targets

```text
first-page screenshot exists
cover media occupies the dominant first-page visual area when media exists
no text overlaps media zones
footer visible and aligned
page count and text extraction pass
forbidden text absent
required sections present
```
