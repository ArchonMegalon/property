# Property Content Publication Approval

Subscribr production output is never publication approval.

## Human Review

A named reviewer must approve:

- script;
- title;
- thumbnail;
- voice;
- visuals;
- disclosure;
- publication destination.

## Publication Gate

Publication is blocked unless all conditions pass:

- source packet validation;
- script validation;
- privacy validation;
- fair-housing validation;
- financial and legal language validation;
- media-rights validation;
- freshness check;
- human approval.

`PROPERTYQUARRY_SUBSCRIBR_DIRECT_PUBLISH_ENABLED` defaults to `0` and should stay disabled until a separate publication subsystem exists.
