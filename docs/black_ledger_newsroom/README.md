# Black Ledger Newsroom

EA owns the assisted editorial and provider-proof support layer for the Black Ledger Newsroom.

This folder keeps the bounded local inputs that EA needs to:

- materialize a newsroom editorial packet from approved sample episode data;
- verify which owned providers are safe candidates for newsroom work;
- fail closed when no photoreal render lane has verified commercial, watermark-free readiness.

Files:

- `SAMPLE_TURN1_EPISODE.yaml`
- `EPISODE_AND_SEGMENT_CONTRACT.yaml`
- `ANCHOR_BIBLE.yaml`
- `MEDIA_FACTORY_RUNTIME_CONTRACTS.yaml`

These files are support inputs only. They do not make EA the truth owner for Black Ledger episodes.
